PPG Monitor — Setup Guide

This app uses the Web Bluetooth API to connect directly to your pulse oximeter from the browser. Requirements:

  • Chrome or Edge on desktop (Windows / macOS / Linux)
  • Served over HTTPS or localhost
  • Bluetooth enabled on your computer

Select a Device Profile matching your oximeter brand before connecting. If your device isn't listed, choose Custom UUIDs and enter the BLE service and characteristic UUIDs from your device's documentation or a BLE scanner app (e.g. nRF Connect).

Packet Parser: Each brand encodes its data differently. The parsers here cover the most common formats. If values look wrong, check the browser console — raw bytes are logged there. You can extend PROFILES in the source.

Recording: Click Record to tag data. Use Save CSV or Save JSON to export — both include timestamps, HR, SpO₂, and raw PPG values.
